# Fridge Helper

## Цель проекта

Помочь пользователю не забывать, какие продукты есть в холодильнике, что нужно купить, и подсказывать, что можно приготовить из имеющихся продуктов.

## Основные возможности

### Холодильник

- **Цель:** Пользователь хочет знать, какие продукты сейчас есть в холодильнике
- **Описание:** Пользователь может добавлять продукты в список «в холодильнике», удалять их и отмечать, что продукт закончился. Продукты отображаются в удобном списке.
- **Критерии приёмки:**
  - Пользователь может добавить продукт с названием и количеством
  - Пользователь может удалить продукт из холодильника
  - Пользователь может отметить продукт как закончившийся
  - Список продуктов отображается актуальным

### Список покупок

- **Цель:** Пользователь хочет вести список продуктов, которые нужно купить
- **Описание:** Пользователь может добавлять продукты в список покупок, отмечать купленное и удалять позиции. Продукты из холодильника, отмеченные как закончившиеся, автоматически добавляются в список покупок.
- **Критерии приёмки:**
  - Пользователь может добавить продукт в список покупок
  - Пользователь может отметить продукт как купленный (он переносится обратно в холодильник)
  - Пользователь может удалить позицию из списка
  - Закончившиеся продукты из холодильника автоматически появляются в списке покупок

### Мои блюда и категории

- **Цель:** Пользователь хочет хранить список блюд, которые он умеет готовить, с разбивкой по категориям
- **Описание:** Пользователь может создавать категории блюд (например, «Завтраки», «Супы», «Паста») и добавлять в них свои рецепты/блюда. У каждого блюда можно указать список необходимых продуктов.
- **Критерии приёмки:**
  - Пользователь может создать категорию блюд
  - Пользователь может добавить блюдо в категорию с названием и списком продуктов
  - Пользователь может редактировать и удалять блюда и категории
  - Категории и блюда отображаются в удобном виде

### Что приготовить?

- **Цель:** Приложение подсказывает пользователю, какие блюда можно приготовить из имеющихся продуктов
- **Описание:** На основе продуктов в холодильнике и списка блюд приложение показывает, какие блюда можно приготовить прямо сейчас (все продукты есть) и какие — с добавлением 1-2 недостающих продуктов.
- **Критерии приёмки:**
  - Приложение показывает блюда, которые можно приготовить из имеющихся продуктов
  - Приложение показывает блюда, для которых не хватает 1-2 продуктов
  - Пользователь видит, каких именно продуктов не хватает

## Дополнительные заметки

- Интерфейс в стиле «Чистый минимализм» — светлый фон, простота, как стандартные iOS-приложения
- Первая версия включает все четыре возможности
- Приложение должно быть простым и интуитивным
